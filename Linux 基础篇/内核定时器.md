# 一、Linux 内核定时器原理
## 1、内核时间管理
1、`Cortec-M`内核使用` Systick` 作为系统定时器。

2、定时器：硬件定时器、软件定时器。

①、硬件定时器提供时钟源，时钟源的频率可以设置， 设置好以后就周期性的产生定时中断，系统使用定时中断来计时；

②、软件定时器依靠系统定时器来驱动。

3、linux 内核频率可以设置，图形化界面配置。

4、`HZ` 表示系统节拍率。



## 2、内核时间管理
Linux 内核使用全局变量 `jiffies` 来记录系统从启动以来的系统节拍数，系统启动的时候会将 `jiffies` 初始化为 0
